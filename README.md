Project Overview

This project is developed as part of the **CodeAlpha Machine Learning Internship**.
The objective of this project is to build a machine learning model that predicts an individual's creditworthiness based on their financial and personal data.

The model helps classify whether a customer is a **good credit risk** or a **bad credit risk** using historical financial information.

---
 Objective

* To preprocess financial dataset
* To build a classification model
* To evaluate model performance using accuracy and other metrics

---

 Dataset Used

**Dataset Name:** German Credit Dataset

The dataset contains information such as:

* Age
* Job
* Housing
* Credit amount
* Duration
* Saving accounts
* Checking accounts
* Purpose of loan

The last column represents the **credit risk category**.

 Steps Performed in the Project
